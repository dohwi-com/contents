## Frontmatter Configuration

The following frontmatter fields are supported:

| Field | Type | Description |
|Ref | Ref | Ref|
| `title` | `string` | The title of the post |
| `date` | `string` | Publication date (YYYY-MM-DD) |
| `description` | `string` | Brief summary for SEO and previews |
| `tags` | `string[]` | Array of tags for categorization |
| `series` | `string` | (Optional) Name of the series this post belongs to |
| `seriesOrder` | `number` | (Optional) Order within the series |
| `ogImage` | `string` | (Optional) Path to Open Graph image |

## Note on Custom Components

Currently, standard Markdown elements are supported. Custom React components are not yet configured in the MDX provider.
