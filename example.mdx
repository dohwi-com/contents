## 기본 텍스트 서식

일반 텍스트는 그냥 작성하면 됩니다. **굵은 글씨**, *기울임*, ~~취소선~~ 모두 지원합니다.

줄바꿈은 빈 줄을 하나 추가하면 새로운 문단이 됩니다.

## 링크와 이미지

[도휘닷컴](https://dohwi.com)처럼 인라인 링크를 사용할 수 있습니다.

이미지도 마크다운 문법 그대로 사용합니다:

```markdown
![대체 텍스트](이미지-URL)
```

## 코드

인라인 코드는 `console.log("안녕하세요")` 이렇게 백틱으로 감쌉니다.

코드 블록은 언어를 지정하면 구문 강조가 적용됩니다:

```typescript
interface Post {
  title: string;
  date: string;
  tags: string[];
}

function formatDate(date: string): string {
  return new Date(date).toLocaleDateString("ko-KR", {
    year: "numeric",
    month: "long",
    day: "numeric",
  });
}
```

```css
.container {
  display: flex;
  align-items: center;
  gap: 1rem;
}
```
